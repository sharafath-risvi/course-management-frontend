# Course Management System - Frontend

## 📌 Overview
This is the **frontend** of the Course Management System built using **React.js**.  
It provides a user-friendly interface to interact with the backend APIs.

This frontend allows users to:
- View courses
- Add courses
- Edit courses
- Delete courses

---

## 🚀 Tech Stack
- React.js
- JavaScript
- HTML
- CSS
- Axios

---

## ✨ Features
- Display all available courses
- Add a new course
- Edit existing course
- Delete course
- Connect with Spring Boot backend
- Responsive user interface
